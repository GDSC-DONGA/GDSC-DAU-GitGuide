# 버전 관리 팁 

> Q : 보통 버전은 어디에 적어요?  
A : 커밋 메시지 제목 부분에 코멘트로 달아준다.

>Q : 이전 버전은 어디서 확인해요?  
A : 특정 파일에 대해 버전 기록을 보고 싶으면 repo에서 해당 파일 클릭 -> 오른쪽 상단 History 클릭 하면 됨

>Q : 혹시 관련 단축키가 있을까요?  
A : 아래 표를 확인해 주세요

|설명|단축키|
|----|----|
|Code 탭 이동|g+c|
|Issues 탭 이동|g+i|
|Pull requests 탭 이동|g+p|
|파일 찾기|t|
|해당 코드 라인 이동|l|
|브랜치 또는 태그 변경|w|
|검색창 이동|s or /|
